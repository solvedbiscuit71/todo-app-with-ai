## Todo App with AI Features

This is my first project with AI features. The goal of this project is to learn how to build AI features on top of traditional SaaS application such as Todo List. Below are the list of SaaS features and AI features that will be implemented as part of this project.

### SaaS Features:
- [ ] Create new todo item
- [ ] Mark todo items as completed
- [ ] Delete todo items

### AI Features:
- [ ] Classify todo items into user-defined categories based on title and description.
- [ ] Assign due date to todo items based on cues from the title and description.
